# Simple Interest Calculator

A calculator to calculate simple interest given principal, annual rate of interest and time period in years.

## Introduction

Simple Interest is a quick method of calculating the interest charge on a loan. It is determined by multiplying the daily interest rate by the principal amount by the number of days that elapse between payments.

## Usage

Run the bash script:

```bash
bash simple-interest.sh
```

## Inputs

- `p` - Principal amount
- `t` - Time period in years
- `r` - Annual rate of interest

## Output

```
simple interest = p*t*r/100
```

## Example

Given:
- Principal: 1000
- Rate: 5%
- Time: 2 years

Simple Interest = (1000 * 5 * 2) / 100 = **100**
